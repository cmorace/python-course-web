Homework 4

This homework covers the topic of iteration. We have seen two kinds of iteration in class, the while loop and the for loop. Each exercise is preceded by a similar example to help you.

1. print()’s end Argument

The print function has a keyword argument end. Whatever value we assign to end will be printed after printing out the previous arguments. Below is an example.

Example: Print the Natural Numbers

[1]:

n = 20
for i in range(1, n):
    print(i, end=",")
print(n)

Print the odd numbers from \(1\) to \(n\) on the same line using a for loop. Can you do it without any if statements?

2. Printing Unicode Characters

Unicode includes technical symbols, punctuations, and many other characters used in writing text. In Python, to print unicode characters we can:

  1. convert a unicode character from hexidecimal to a base-10 integer with the int(hex_code, 16) function.

  2. then convert the integer to a character using the chr() function.

Below, is an example.

Example: Printing Bopomofo

In the example below ㄅ is represented by "3105" and ㄬ is represented by "312C". Other unicode character codes can be looked up in a chart here.

[2]:

low = int("3105", 16)
high = int("312C", 16)
for i in range(low, high+1):
    print(chr(i), end="")

3. Math Quiz with randint()

Python has a built-in package named random which has many functions for generating random numbers. The function randint(a, b) will return a random integer between a and b.

Example: Division Quiz

[1]:

from random import randint
while True:
    a = randint(2, 10)
    b = randint(2, 10)
    product = a * b
    answer = input("What is "+str(product)+"/"+str(a)+"?")
    if int(answer) == b:
        print("very good")
    else:
        print("sorry, wrong answer")

    continue_response = input("press q to quit:")
    if continue_response == "q":
        break

4. Calculating the Sum of Natural Numbers

A useful formula for the sum of natural numbers from \(1\) to \(n\) is:

\[1+2+3+\dots+n=\frac{n(n+1)}{2}\]

Write a loop to verify this formula.

Example: While Loop Implementation

[5]:

i = 1
n = 100
sum_n = 0
while i <= n:
    sum_n += i
    i += 1
print(sum_n)
print(sum_n == n*(n+1)/2)

5050
True

5. Summing to \(\pi\)

Example: The Bailey–Borwein–Plouffe Formula

The The Bailey–Borwein–Plouffe Formula was discovered in 1995 by Simon Plouffe. It states that

\[\pi = \sum_{k=0}^{\infty} \left(\frac{1}{16^k}\left(\frac{4}{8k+1}-\frac{2}{8k+4}-\frac{1}{8k+5}-\frac{1}{8k+6}\right)\right)\]

Remarkably, this formula computes the first 15 digits of pi in just 10 iterations.

[6]:

sum = 0
for k in range(10):
    _8k = 8*k
    sum += (4/(_8k+1) - 2/(_8k+4) - 1/(_8k+5) - 1/(_8k+6)) / (16**k)
print(sum)

3.1415926535897913

Exercise: The Leibniz Formula

The Leibniz formula was discovered by the Indian Mathematician Madhava of Sangamagrama in the 14th century. It states that:

\[1 - \frac{1}{3} + \frac{1}{5} - \frac{1}{7} + \frac{1}{9} \dots = \frac{\pi}{4}\]

Use a for loop to calculate this sum and multiply the final sum by 4 to check how close the result is to \(\pi\).
