Welcome to the Python Course Website!

Hi there. This page includes some resources for our Python course. Check it out!

* `Programming Environment Setup <environment_setup.rst>`_ has instructions for setting up a python programming environment.
* `Homework <homework.rst>`_ posts the assigned homework.
* `Notes <notes.rst>`_ contains some notes about the topics we cover in class.
* `Tutorials <tutorials.rst>`_ includes tutorials for the programming tools we use.

.. note::
   Once you have programming environment setup, stay up to date with the latest version of pycat by running the following command in your terminal

   .. code:: bash

     pip install git+https://bitbucket.org/dwhite0/pycat.git
